# CatalogoCursosCSharp
Este proyecto es una mini aplicación llamada “Catálogo de Cursos” que permite listar, buscar y paginar cursos de ejemplo.
 El objetivo es practicar la coordinación de tareas mediante el uso de Trello y GitHub, asignando responsables, fechas y seguimiento de avance.
 Propósito del trabajo
El proyecto forma parte del caso práctico de “Coordinación de tareas: asignación, seguimiento y plazos”, cuyo propósito es demostrar:

1. Desglose de tareas con Definition of Done (DoD).

2. Asignación de responsables y fechas.

3. Uso de tablero (Trello) y GitHub para control de versiones.

4. Comunicación del avance mediante un mini reporte.

Tecnologías utilizadas:

Lenguaje: C#

Control de versiones: Git y GitHub

Gestión del proyecto: Trello

Editor: Visual Studio Code

 Funcionalidades principales
Muestra una lista estática de 3 a 5 cursos.

Permite buscar cursos por nombre (filtro).

Incluye paginación simulada o filtrado básico.

Código documentado y comentado.
